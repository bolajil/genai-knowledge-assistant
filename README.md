# VoultMIND Knowledge Assistant

VoultMIND is an AI-powered knowledge assistant that helps users access and query their documents efficiently. It provides document indexing, retrieval, and summarization capabilities.

## Features

- Document indexing with vector embeddings
- Semantic search for relevant document retrieval
- AI-powered document summarization and question answering
- Support for multiple document formats
- Interactive agent assistant for research tasks
- AWS documentation capabilities

## Installation

1. Clone this repository
2. Install dependencies: pip install -r requirements.txt
3. Create a .streamlit/secrets.toml file with your API keys
4. Run the application: python main.py

## Usage

The application provides multiple tabs for different functionalities:

- **Query Assistant**: Ask questions about your documents
- **Document Ingestion**: Upload and index new documents
- **Agent Assistant**: Get research help and document summaries
- **Enhanced Research**: Combine multiple sources for in-depth analysis
